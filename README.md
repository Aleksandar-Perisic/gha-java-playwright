# GitHub Actions in action - Playwright Java Tests

This repository contains **end-to-end UI tests** using [Playwright](https://playwright.dev/) with **Java** and **JUnit 5**.

## 🚀 Getting Started

### 1️⃣ Prerequisites

- Java 17+
- Maven

### 2️⃣ Install Dependencies

```sh
mvn install
```

### 3️⃣ Run Tests

```sh
mvn test
```
